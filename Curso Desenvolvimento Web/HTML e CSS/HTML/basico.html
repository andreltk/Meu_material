Toda página HTML necessita de Tags básicas pra funcionar, entre elas está:


  • !DOCTYPE
  • html
  • head
  • title
  • body

DOCTYPE

<!DOCTYPE>

Define para o navegar o tipo e versão do HTML, no caso do HTML 5, basta adicionar !DOCTYPE html, lembrando que a tag não é fechada

HTML

<html>

Defini a raíz da página e deve ser fechada ao final da mesma, além disso pode conter o atributo lang, como em lang="pt-BR", de modo a dizer o idioma da mesma, facilitando para bot e softwares de acessibilidade

HEAD

<head>

Serve para adicionar metadados sobre a página, dos quais o meta charset, que indica qual codificação será usada na página, UTF-8 por exemplo. Pode incluir a Tag Title que define um título pra página

BODY

<body>

É o corpo da página em si, todo conteúdo será apresentado
