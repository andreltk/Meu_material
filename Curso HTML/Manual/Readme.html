Manual HTML

No HTML são muito usado as Tags (marcações)

HTML tags são elementos cercados por < e > (Parênteses Angulares):

<nomedatag> conteúdo... </nomedatag>

Tags normalmente vem em pares, como em <p> e </p> A primeira é chamada tag de abertura e a segunda tag de fechamento

As marcações básicas são:

Tag Descrição
<!DOCTYPE> Define o tipo do Documento
html Define um Documento HTML
head Define informação sobre um Documento
title Define o título de um documento
body Define o corpo do documento
h1 - h6 Define um cabeçalho
p Insere um parágrafo
br Insere uma quebra de linha
hr Faz uma mudança temática no conteúdo
!--...-- Define um comentário
